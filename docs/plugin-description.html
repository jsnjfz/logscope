V2EX LogScope is an IntelliJ IDEA plugin that presents V2EX content inside a log-style tool window so you never have to leave the IDE.
V2EX LogScope 是一款 IntelliJ IDEA 插件，通过日志风格的工具窗口在 IDE 内隐蔽浏览 V2EX 话题。

Key Features
主要特性

  • Stealth log-themed UI with MASK toggle
    隐匿日志风格界面，并提供 MASK 切换
  • Browse different nodes（Hot/Tech/Creative/Play/Hot Topics/All 等）
    支持热门、技术、创意、好玩、最热、全部等节点切换
  • View topic summaries, details and replies with pagination
    支持话题摘要、详情与回复分页查看
  • Proxy configuration and API Token management
    内置代理与 API Token 配置
  • Keyboard-friendly controls and compact layout
    紧凑布局，交互体验贴近终端日志

Technical Features
技术特点

  • Uses V2EX API v2
    使用 V2EX API v2
  • Supports HTTP/SOCKS proxy
    支持 HTTP/SOCKS 代理
  • Simple layout focused on readability
    以易读性为优先的简洁布局

What's New (0.0.4)
更新说明（0.0.4）

  • Added inline reply composer with API v2 + legacy fallback
    新增 IDE 内回复能力，并对 API 2.0/旧版接口自动回退
  • Reply composer is hidden behind a toolbar toggle for better readability
    回复框可通过工具栏按钮按需展开，保持日志视图紧凑
  • Fixed localized labels that previously showed as garbled text
    修复多处中文注释与文案乱码问题
